# body-motion-game
Real-time bodymotion game using python, django and computer vision , an interactive motion-controlled game that tracks body movements in real time, a gesture-based gaming sysytem using real-time body tracking technology

## Technologies Used

- Python
- Django
- OpenCV
- MediaPipe
- HTML
- CSS
- JavaScript
- SQLite
- VS Code
- Real-time body motion game developed using Python, Django, OpenCV, and MediaPipe with interactive web-based gameplay and motion tracking.

  ## Features

- Real-time body motion detection
- Camera-based game control
- Interactive gameplay experience
- Human body tracking using computer vision
- Fast response and movement recognition
- User-friendly web interface
- Real-time motion processing
- Dynamic player interaction
- Lightweight and responsive system
- Browser-based accessibility
  
  ## Key Features
  
- Real-time motion tracking
- Gesture-based game interaction
- Computer vision integration
- Web-based gaming interface
- Responsive gameplay system

## How to Run the Project

### 1. Clone the Repository

```bash
git clone <your-repository-link>
```

### 2. Navigate to the Project Folder

```bash
cd body-motion-game
```

### 3. Create a Virtual Environment

```bash
python -m venv venv
```

### 4. Activate the Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### 5. Install Required Packages

```bash
pip install -r requirements.txt
```

### 6. Run Database Migrations

```bash
python manage.py migrate
```

### 7. Start the Django Server

```bash
python manage.py runserver
```

### 8. Open in Browser

```text
http://127.0.0.1:8000/
```

### 9. Allow Camera Access

Make sure your webcam is connected and allow browser camera permissions for body motion detection.
